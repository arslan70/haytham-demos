## Run this on your own idea

These reports were generated by [Haytham](https://github.com/arslan70/haytham), a free Claude Code plugin. One command, ~20 minutes, no API keys.

```
/plugin install haytham@haytham
/haytham "your startup idea here"
```

It runs market research, competitor analysis, and gives you a GO/PIVOT/NO-GO verdict backed by evidence. If it says NO-GO, it tells you why. That's the point.
